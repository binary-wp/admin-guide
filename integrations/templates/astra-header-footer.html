Header & Footer

The header and footer appear on every page. They are managed through the Astra theme Customizer ({{astra_settings_link}}) — not in Elementor or the block editor.

Header

The header is configured in Appearance > Customize > Header Builder. Astra provides a drag & drop header builder with multiple rows:

Above Header Row

Optional utility row above the main header — used for announcements, secondary navigation, or contact info.

Primary Header Row

  • Site Logo — managed in Customizer > Header Builder > Site Identity
  • Primary Menu — managed at {{wp_menus_link}} or in the Customizer
  • Search — optional search widget in the header

Footer

The footer is configured in Appearance > Customize > Footer Builder:

Primary Footer Row

  • Footer navigation menu — managed at {{wp_menus_link}}
  • Footer widget areas — managed in {{wp_widgets_link}}

Below Footer Row

  • Copyright notice — configured in Customizer > Footer Builder > Copyright element
